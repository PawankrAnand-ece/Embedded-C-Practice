# 21 - UART Message Transmit

## Objective
Simulate UART string transmission using character-by-character transfer.

## Concept
UART communication sends one byte at a time.
This program transmits a complete message using pointer traversal.

## Example
Message:
System Boot Successful

## Industrial Use
- Debug logging
- Boot messages
- Sensor data transmission
- GSM/GPS communication
- Serial console output

## Output
System Boot Successful
