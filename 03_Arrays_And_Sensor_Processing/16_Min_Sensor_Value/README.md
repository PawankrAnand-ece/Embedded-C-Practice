# 16 - Find Minimum Sensor Value

## Objective
Find the lowest value from sensor samples.

## Concept
Each sensor reading is compared continuously to track the minimum value.

## Example Data
412, 398, 450, 425, 470, 430, 455, 440

## Output
Minimum Sensor Value: 398

## Industrial Use
- Low battery monitoring
- Undervoltage protection
- Minimum temperature tracking
- Dry soil detection
