# 19 - Sensor Threshold Compare

## Objective
Compare sensor value with a predefined threshold.

## Concept
If sensor value crosses the threshold, firmware generates an alert condition.

## Example
Sensor Value: 650  
Threshold: 600  
Result: Alert

## Industrial Use
- Fire alarm trigger
- Smoke sensor monitoring
- Battery low/overvoltage detection
- Temperature protection
- Soil moisture alert

## Output
Sensor Value: 650  
Alert: Threshold crossed
