# 15 - Find Maximum Sensor Value

## Objective
Find highest value from sensor samples.

## Concept
Sensor arrays are scanned one by one and compared to track the maximum reading.

## Example Data
412, 398, 450, 425, 0, 479, 430, 455, 440, 451

## Output
Maximum Sensor Value: 479

## Industrial Use
- Peak temperature detection
- Over-voltage monitoring
- Smoke spike detection
- Maximum current monitoring
