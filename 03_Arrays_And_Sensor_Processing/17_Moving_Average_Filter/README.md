# 17 - Moving Average Filter

## Objective
Smooth sensor readings using a moving average filter.

## Concept
A moving average filter calculates the average of a fixed number of recent samples.

## Example
Window Size: 3

Samples:
500, 520, 510

Average:
510

## Industrial Use
- ADC noise filtering
- Temperature sensor smoothing
- Battery voltage monitoring
- Soil moisture stabilization
- Smoke sensor signal smoothing

## Output
Moving Average Output:  
Window 1 Average: 510  
Window 2 Average: 520  
Window 3 Average: 518  
Window 4 Average: 516  
Window 5 Average: 515  
Window 6 Average: 513
