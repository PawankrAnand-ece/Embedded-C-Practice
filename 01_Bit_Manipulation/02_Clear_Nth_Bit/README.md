# 02 - Clear Nth Bit in Register

## Objective
Clear any selected bit in an 8-bit register using bitwise AND with inverted mask.

## Formula Used
reg = reg & ~(1 << n)

## Explanation
A mask is generated by shifting binary 1 to the target bit position.
The mask is inverted so that only the target bit becomes 0 and all other bits remain 1.
AND operation clears only the selected bit.

## Example
Initial Register : 0xFF  
Bit Position     : 3  
Final Register   : 0xF7

## Industrial Use
- GPIO output LOW
- Relay OFF
- Buzzer OFF
- Clear alarm/fault flag
- Disable peripheral control bit

## Output
Before clearing bit: 0xFF  
After clearing bit 3: 0xF7
