# 01 - Set Nth Bit in Register

## Objective
Set any selected bit in an 8-bit register using bitwise OR operation.

## Formula Used
reg = reg | (1 << n)

## Explanation
A mask is generated by shifting binary 1 to the required bit position.
OR operation then makes only that target bit HIGH.

## Example
Initial Register : 0x00  
Bit Position     : 3  
Final Register   : 0x08

## Industrial Use
- GPIO output HIGH
- Relay enable
- Alarm flag set
- Peripheral control register enable

## Output
Before setting bit: 0x00
After setting bit 3: 0x08
