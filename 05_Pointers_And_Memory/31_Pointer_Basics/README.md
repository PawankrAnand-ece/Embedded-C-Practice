# 31 - Pointer Basics

## Objective
Understand basic pointer usage in Embedded C.

## Concept
Pointers store memory addresses.

## Operators

&  → address of variable  
*  → access value using address

## Working
- Variable created
- Pointer stores variable address
- Pointer accesses variable value

## Industrial Use
- Hardware register access
- Peripheral drivers
- DMA buffers
- Embedded firmware development
