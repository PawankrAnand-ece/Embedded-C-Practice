# 36 - Pointer To Structure

## Objective
Understand structure pointers in Embedded C.

## Concept
Structures group multiple related variables into one datatype.

Pointers allow efficient access to structure members.

## Operators

.   → direct structure access  
->  → structure pointer access

## Working
- Structure created
- Pointer stores structure address
- Members accessed using arrow operator

## Industrial Use
- Sensor drivers
- UART drivers
- HAL libraries
- RTOS objects
- Peripheral configuration

## Output
Sensor information displayed using structure pointer.
