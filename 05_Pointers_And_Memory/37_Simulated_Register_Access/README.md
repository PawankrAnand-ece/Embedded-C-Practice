# 37 - Simulated Register Access

## Objective
Simulate microcontroller register access using pointers.

## Concept
Embedded peripherals are controlled using memory mapped registers.

Firmware accesses registers using pointers.

## Important Keyword
volatile

Used because hardware registers may change unexpectedly.

## Operations
- Set register bit
- Clear register bit

## Industrial Use
- GPIO control
- UART registers
- Timer registers
- ADC configuration
- Embedded firmware development

## Output
Register bits controlled using pointer operations.
