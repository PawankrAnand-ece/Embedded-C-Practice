# 34 - Swap Values Using Pointers

## Objective
Swap two variable values using pointers.

## Concept
Pointers allow direct modification of original variables using memory addresses.

## Working
- Variable addresses passed to function
- Function accesses original memory
- Values swapped using temporary variable

## Operators

&  → address of variable  
*  → access value using pointer

## Industrial Use
- Sensor sorting
- Buffer management
- DMA handling
- Embedded data processing

## Output
Sensor values swapped successfully.
