# 32 - Pointer Arithmetic

## Objective
Understand pointer increment and memory traversal.

## Concept
Pointer arithmetic moves pointer according to datatype size.

Example:
uint16_t pointer increments by 2 bytes.

## Working
- Pointer assigned to array base address
- Pointer incremented using ptr++
- Values accessed sequentially

## Industrial Use
- DMA buffers
- UART parsing
- Sensor frame processing
- Embedded protocol handling

## Important Concept
Array name represents base address of array.
